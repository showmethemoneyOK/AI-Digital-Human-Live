## 🚀 快速启动指南

### 步骤1：环境准备
```bash
# 1. 克隆或解压项目
cd ai-digital-human-live

# 2. 运行部署脚本
deployment\setup.bat

# 3. 根据提示选择平台
```

### 步骤2：配置API密钥
```bash
# 编辑配置文件
notepad config\config.json

# 填入以下API密钥：
# TikTok: OpenAI, ElevenLabs, D-ID, TikHub
# 抖音: 火山方舟, 阿里云, 硅基智能, 抖音开放平台
```

### 步骤3：安装必要工具
- **TikTok**: 安装OBS Studio，配置推流
- **抖音**: 安装抖音直播伴侣，登录账号

### 步骤4：启动系统
```bash
# 启动直播系统
start_live.bat

# 查看日志
tail -f logs\ai_live_*.log
```

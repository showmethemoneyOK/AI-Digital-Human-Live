### 2.2 时序图
```mermaid
sequenceDiagram
    participant T as TikTok Platform
    participant C as Comment Collector
    participant A as AI Processor
    participant S as TTS Service
    participant D as Digital Human
    participant O as OBS Streamer
    participant L as Live Stream

    Note over T,L: 直播开始
    O->>L: 开始RTMP推流
    L->>T: 直播画面传输
    
    loop 每500ms
        T->>C: 推送新弹幕
        C->>A: 发送弹幕文本
        A->>A: 清洗+过滤敏感词
        A->>A: LLM生成回复
        A->>S: 发送回复文本
        S->>D: 返回音频文件
        D->>D: 生成数字人视频
        D->>O: 更新视频源
        O->>L: 推流更新后的视频
        L->>T: 观众看到AI回复
    end
    
    Note over T,L: 直播结束
    O->>L: 停止推流
```

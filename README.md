## 🎯 项目概述

这是一个完整的AI数字人实时互动带货直播系统，支持**抖音（国内版）**和**TikTok（国际版）**双平台。系统采用100%官方合规方案，确保长期稳定运行。

### 抖音合规方案
- **弹幕监听**: 抖音开放平台官方WebSocket API
- **AI对话**: 国产大模型（通义/豆包/DeepSeek等）
- **TTS语音**: 阿里云/腾讯云/讯飞
- **数字人**: 硅基/有料/其他抖音白名单厂商
- **推流**: 抖音直播伴侣（唯一合规推流）

### TikTok合规方案
- **弹幕监听**: TikTok Developer API / Webhook（官方）
- **AI对话**: 通义/豆包/DeepSeek/GPT
- **TTS语音**: 阿里云/讯飞/ElevenLabs
- **数字人**: 硅基/有料/HeyGen/Synthesia
- **推流**: OBS Studio + TikTok官方RTMP推流地址
- **强制标注**: 画面/标题必须标 AI Generated / Virtual Host

## 🚀 快速开始

### 1. 环境要求
- Python 3.9+
- Windows 10/11（抖音必须）
- 网络：上传带宽 ≥ 8Mbps
- 硬件：i5/Ryzen5+，8GB+ RAM

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置平台
#### 抖音配置
1. 注册抖音开放平台账号
2. 申请直播权限
3. 获取APP_ID和APP_SECRET
4. 配置白名单数字人服务

#### TikTok配置
1. 注册TikTok Developer账号
2. 申请Live Streaming/Webhook权限
3. 获取API密钥
4. 配置OBS推流

### 4. 运行系统
```bash
# 抖音模式
python src/main.py --platform douyin

# TikTok模式
python src/main.py --platform tiktok
```

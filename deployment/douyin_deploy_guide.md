# 抖音AI数字人直播部署指南

## 一、前置条件

### 1.1 账号准备
- ✅ 抖音账号（已实名认证）
- ✅ 抖音直播权限（粉丝≥1000）
- ✅ 抖音开放平台企业账号

### 1.2 硬件要求
- Windows 10/11 操作系统（必须）
- CPU：i5/Ryzen5 或更高
- 内存：8GB 或更高
- 网络：上传带宽 ≥ 8Mbps
- 存储：10GB 可用空间

### 1.3 软件要求
- Python 3.9+
- 抖音直播伴侣（最新版）
- Git（可选）

## 二、抖音开放平台配置

### 2.1 注册开放平台
1. 访问 [抖音开放平台](https://open.douyin.com/)
2. 使用企业资质注册账号
3. 完成实名认证

### 2.2 创建应用
1. 进入控制台 → 应用管理
2. 点击"创建应用"
3. 选择"自用型应用"
4. 填写应用信息：
   - 应用名称：AI数字人直播助手
   - 应用描述：AI数字人直播互动系统
   - 应用分类：工具类

### 2.3 申请权限
1. 进入应用详情 → 能力管理
2. 申请以下权限：
   - **直播权限**（必须）
   - 用户信息（可选）
   - 内容发布（可选）
3. 提交审核（1-3个工作日）

### 2.4 获取凭证
审核通过后，获取：
- **App ID**：应用唯一标识
- **App Secret**：应用密钥（妥善保管）
- **Access Token**：通过API获取

## 三、数字人服务配置

### 3.1 选择白名单厂商
抖音合规要求必须使用白名单厂商：
- ✅ 硅基智能
- ✅ 有料AI
- ✅ 相芯科技
- ✅ 其他抖音认证厂商

### 3.2 注册数字人服务
1. 访问厂商官网注册账号
2. 完成企业认证
3. 创建数字人形象
4. 获取API密钥

### 3.3 配置数字人
1. 选择2D/3D数字人形象
2. 配置语音合成参数
3. 设置口型同步
4. 测试推流输出

## 四、AI服务配置

### 4.1 选择AI模型（三选一）

### 4.2 获取API密钥

## 五、TTS服务配置

### 5.1 选择TTS服务
#### 国内服务（推荐）
- 阿里云智能语音交互
- 腾讯云语音合成
- 讯飞开放平台

#### 国际服务

### 5.2 配置参数
- 语音类型：选择适合带货的音色
- 语速：1.0-1.2倍
- 音调：适中
- 音量：100%

## 六、系统部署

### 6.1 环境准备
```bash
# 1. 克隆项目
git clone https://github.com/showmethemoneyOK/AI-Digital-Human-Live.git
cd AI-Digital-Human-Live

# 2. 创建虚拟环境
python -m venv venv

# 3. 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt
```

### 6.2 配置文件
```bash
# 1. 复制配置文件
cp config/douyin_config_example.json config/config.json

# 2. 编辑配置文件
# 使用文本编辑器打开 config/config.json
# 填入你的API密钥和配置
```

### 6.3 环境变量（可选）
创建 `.env` 文件：
```env
# 抖音配置
DOUYIN_APP_ID=your_app_id
DOUYIN_APP_SECRET=your_app_secret

# AI配置
AI_API_KEY=your_ai_api_key

# TTS配置
TTS_API_KEY=your_tts_api_key

# 数字人配置
DIGITAL_HUMAN_API_KEY=your_digital_human_api_key
```

## 七、直播伴侣配置

### 7.1 安装直播伴侣
1. 从抖音官网下载最新版直播伴侣
2. 安装到默认路径
3. 使用抖音账号登录

### 7.2 配置推流
1. 打开直播伴侣
2. 点击"开播" → "电脑直播"
3. 选择"游戏进程"或"窗口捕获"
4. 捕获数字人软件窗口

### 7.3 画面设置
- 分辨率：1080×1920（竖屏）
- 帧率：30fps
- 码率：4000kbps
- 编码：H.264

## 八、系统启动

### 8.1 启动顺序
```bash

# 3. 启动AI直播系统

```

### 8.2 验证连接

- **画面标注**：必须在画面明显位置标注"AI数字人主播"

## 十一、故障排除

### 11.1 常见问题
#### Q1: 抖音API连接失败
- 检查App ID和Secret是否正确
- 确认权限已通过审核
- 检查网络连接

#### Q2: 数字人画面不显示
- 检查数字人软件是否运行
- 确认虚拟摄像头驱动正常
- 检查直播伴侣捕获设置

#### Q3: AI回复延迟高
- 检查网络延迟
- 降低AI模型复杂度
- 优化提示词长度

#### Q4: 直播中断
- 检查网络稳定性
- 确认API密钥未过期
- 查看系统资源使用情况

### 11.2 紧急处理
1. **立即切换**：切换到真人主播
2. **系统重启**：重启AI直播系统
3. **降级模式**：使用简单回复模板
4. **联系支持**：联系技术服务商
